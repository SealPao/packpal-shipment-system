# Windows App UI Flow

## 1. Login Screen
- 顯示系統名稱
- 顯示工作站代號
- 輸入或掃描員工編號
- 顯示版本號
- 顯示底部作者資訊

## 2. Mode Selection Screen
- 出貨作業
- 維修收貨
- 退貨收貨
- 切換作業人員
- 結束工作

## 3. Shipment Screen
- 顯示出貨單號
- 顯示員工資訊
- 顯示工作站
- 顯示時間
- 顯示物流按鈕
- 掃描宅配單號後結束

## 4. Repair Screen
- 顯示宅配單號
- 顯示員工資訊
- 顯示時間
- 掃描設備序號（可選）
- 完成開箱

## 5. Document Capture Screen
- 拍攝送修文件
- 自動文件校正
- 顯示校正結果
- 存檔

## 6. Return Screen
- 顯示宅配單號
- 顯示員工資訊
- 完成開箱
- 掃描或輸入銷貨單號
- 存檔
